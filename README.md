# Mini Endless Runner

A simple 2D endless runner game built with HTML, CSS, and JavaScript.

## How to play

- Open `index.html` in your browser.
- Press `Space` or `Arrow Up` to start and jump.
- Avoid the red obstacles.
- The score increases over time.

## Files

- `index.html` - game canvas and UI
- `style.css` - visual styling
- `script.js` - game logic and animation
